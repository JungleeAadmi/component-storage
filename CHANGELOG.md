# Changelog

All notable changes to Component Storage will be documented in this file.

## [1.0.0] - 2025-11-20

### Added
- Initial release
- Component CRUD operations
- Storage container management (Type 1, Type 2, Custom)
- Global search with Ctrl+K shortcut
- Dashboard with statistics
- Low stock alerts
- Image upload support
- Auto-generated tray addresses
- Visual storage grid with color coding
- Component library with filtering
- Auto-backup system (daily at midnight)
- One-liner installation script
- Update and uninstall scripts
- Systemd service integration
- Dark mode UI
- Responsive design

### Features
- SQLite database for data persistence
- Express REST API
- React frontend with Vite
- TailwindCSS styling
- shadcn/ui components
- Image storage and serving
- Database migrations
- Comprehensive documentation
